# Quick Start — Up and Running in 10 Minutes

Parent: [[CLAUDE]]

---

## What You Need

- Claude desktop app (claude.ai/download)
- Claude Pro or Max plan ($20–$100/month)
- Cowork mode enabled (beta feature — enable in Claude settings)

That's it. No database. No automation. No voice platform. Those come later if you want them.

---

## Step 1 — Install the Skills (2 minutes)

Skills are the AI shortcuts that make SHINE work.

1. Find your Claude skills folder:
   - **Mac:** `~/.claude/skills/`
   - **Windows:** `%APPDATA%\Claude\skills\`
2. Create two folders: `shine-reports` and `shine-setup`
3. Copy the SKILL.md files into those folders
5. Restart Claude desktop

---

## Step 2 — Add the Workspace (1 minute)

1. Open Claude desktop → Cowork mode
2. Click "Select Folder"
3. Select the `shine-os-cowork` folder
4. Claude now has access to all your workspace files

---

## Step 3 — Run the Setup Wizard (5–7 minutes)

In Cowork, type: `/shine-setup`

The wizard will ask you about your business, name your agents, and customise everything.

---

## Step 4 — Run Your First Session

You're live. Try:

> "Let's work on [the biggest problem you mentioned in setup]. Start a session report and let's make some progress."
