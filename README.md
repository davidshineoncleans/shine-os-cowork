## Get Started (60 seconds)

1. **Download this folder** — click the green `Code` button above → `Download ZIP` → unzip it
2. **Open Claude** — desktop app or claude.ai
3. **Select the folder** — in Cowork mode, select the `shine-os-cowork` folder as your workspace
4. **Say this:**

> *"I've just installed SHINE OS. Let's get set up."*

That's it. Claude reads your `CLAUDE.md` automatically and becomes your AI operations team.
